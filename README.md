# backend-projeto-integrador

## Instalação
```bash
# Clone o repositório
## Via ssh (recomendado)
git clone git@github.com:afmireski/garchop-api.git

## Via HTTPS
git clone https://github.com/afmireski/garchop-api.git

# Acesse o projeto
cd /garchop-api

# Execute o gerenciador de versão
mise trust

# Suba a .env
mise env

# Suba a API
go run ./cmd/main.go
```
